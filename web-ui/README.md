# Aura Integral Solver 2026

Dự án tốt nghiệp: Hệ thống hỗ trợ giải tích phân thông minh với giao diện Liquid Glass 2026.

## Tính năng chính
- **Giải tích phân ký hiệu**: Hỗ trợ các hàm cơ bản (x, sin, cos, exp...).
- **Giao diện Liquid Glass**: Hiệu ứng kính mờ, mesh gradient động, và các khối lơ lửng.
- **Các bước giải chi tiết**: Hiển thị logic giải bài thông qua các bước trung gian.
- **Lịch sử tra cứu**: Lưu lại các bài đã giải trong phiên làm việc.
- **Responsive Design**: Tối ưu hóa trải nghiệm trên cả Mobile và Desktop.

## Công nghệ sử dụng
- **React 18** + **Vite**
- **Framer Motion**: Hệ thống chuyển động và animation cao cấp.
- **Lucide React**: Bộ icon modern.
- **KaTeX**: Hiển thị công thức toán học sắc nét.
- **MathJS**: Xử lý logic biểu thức.

## Cách chạy dự án
1. `npm install`
2. `npm run dev`
