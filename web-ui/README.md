# integral-models
Ứng dụng các thuật toán Machine Learning kết hợp hệ luật trong bài toán hỗ trợ giải tích phân.
Phạm vi bài toán tập trung vào các tích phân đơn một biến, bao gồm các dạng cơ bản như đa thức, lượng giác, hàm mũ – logarit, tích phân dạng hợp (đổi biến), tích phân từng phần đơn giản và phân thức hữu tỉ bậc thấp. Các bài toán được giới hạn ở mức tối đa hai thành phần trong biểu thức, nhằm đảm bảo có thể xác định nhanh phương pháp giải.
